# SKILL - SUPABASE SEGURO

## Objetivo
Cuidar la conexión y operaciones con Supabase.

## Reglas
- No cambiar URL ni anon key salvo que el usuario lo pida.
- No cambiar nombres de tablas.
- No cambiar nombres de columnas.
- No borrar datos.
- No modificar RLS sin explicación.
- Revisar campos reales antes de insertar.
- Login usa tabla usuarios con username, password, rol y activo.
