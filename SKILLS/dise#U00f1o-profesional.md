# SKILL - DISEÑO PROFESIONAL

## Objetivo
Que BIANTI se vea como una tienda real y un sistema serio.

## Reglas visuales
- Sin emojis.
- Usar íconos SVG.
- Mejor contraste.
- Más aire entre elementos.
- Botones consistentes.
- Cards con sombra suave.
- Bordes finos.
- Colores vivos pero elegantes.
- Nada demasiado brillante.
- Nada infantil.
- Nada desordenado.
