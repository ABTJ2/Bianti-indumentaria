# AGENTE 1 - ANÁLISIS GENERAL DEL PROYECTO

## Objetivo
Revisar todo el proyecto BIANTI Indumentaria antes de modificar código.

## Responsabilidades
- Analizar estructura de archivos.
- Revisar rutas de HTML, CSS y JavaScript.
- Detectar scripts rotos o mal conectados.
- Detectar IDs usados por JavaScript.
- Detectar clases importantes usadas por CSS o JS.
- Detectar errores visuales graves.
- Detectar errores de Supabase sin modificar tablas.
- Detectar archivos vacíos o incompletos.
- Detectar etiquetas HTML mal cerradas.
- Detectar estilos duplicados.

## Archivos a revisar
- index.html
- admin/*.html
- assets/css/bianti.css
- assets/css/admin.css
- assets/js/*.js
- assets/js/admin/*.js

## Reglas
- No modificar nada hasta terminar el análisis.
- No cambiar nombres de tablas de Supabase.
- No eliminar IDs ni clases sin saber dónde se usan.
- No hacer cambios visuales todavía.
- Solo entregar un diagnóstico claro.

## Entrega esperada
Al finalizar, entregar:
1. Problemas encontrados.
2. Archivos afectados.
3. Riesgos.
4. Orden recomendado de modificación.
