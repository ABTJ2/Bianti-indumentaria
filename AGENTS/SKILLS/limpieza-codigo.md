# SKILL - LIMPIEZA DE CÓDIGO

Aplicar cuando se modifique HTML, CSS o JavaScript.

## Reglas
- Ordenar HTML.
- Corregir etiquetas mal cerradas.
- Eliminar estilos inline cuando sea posible.
- Evitar duplicar CSS.
- Mantener nombres claros.
- No eliminar IDs usados por JS.
- No mezclar lógica con diseño.
- No crear código innecesariamente complejo.
