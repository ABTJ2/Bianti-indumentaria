# SKILL - GRÁFICOS PROFESIONALES

## Objetivo
Mejorar gráficos de métricas, tracking y contabilidad.

## Reglas
- Usar Chart.js o ApexCharts.
- Mejorar colores.
- Mejorar tooltip.
- Mejorar leyenda.
- Mejorar ejes.
- Mejorar altura del contenedor.
- Mostrar estado vacío si no hay datos.
- Usar formato moneda donde corresponda.
- No inventar datos.
