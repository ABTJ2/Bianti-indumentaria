# GUIA VISUAL BIANTI INDUMENTARIA

## Regla principal
Las imágenes de esta carpeta son referencias visuales.
OpenCode debe replicar el diseño con código real: HTML, CSS y JavaScript.

## Estilo general aprobado
- Diseño moderno, limpio, premium y elegante.
- Desktop 16:9 como referencia principal.
- Sidebar oscuro.
- Topbar oscuro.
- Área principal clara.
- Color de acento violeta.
- Cards blancas con bordes redondeados.
- Sombras suaves.
- Tablas limpias.
- Modales profesionales.
- Mucho espacio visual.
- Tipografía limpia y jerarquía clara.

## Catálogo público
- Header oscuro y limpio.
- Hero visual elegante.
- Buscador y chips de categorías prolijos.
- Tarjetas de producto modernas.
- Footer oscuro con información de contacto.
- Botones de WhatsApp en violeta.

## Login
- Elegante y premium.
- Fondo oscuro.
- Logo BIANTI claro.
- Card central clara o suave.
- Botón principal violeta.
- Sensación de panel privado, no login genérico.

## Sidebar admin definitivo
Orden exacto:
1. Panel principal
2. Métricas
3. Categorías
4. Pedidos
5. Contabilidad
6. Estadísticas

No agregar:
- Configuración
- Backups
- Tracking
- Clientes
- Consultas
- Productos
- Nuevo producto

## Sidebar visual
- Fondo negro o gris muy oscuro.
- Logo BIANTI grande arriba.
- Link activo con fondo violeta.
- Tarjeta de soporte.
- Botón Cerrar sesión abajo.
- Puede tener estado contraído si no rompe la interfaz.

## Estados visuales
Pedidos:
- en_revision: violeta
- vendido: verde
- no_vendido: rojo

Categorías:
- visible sí: verde
- visible no: gris
- usa_talles sí: verde
- usa_talles no: gris

Finanzas:
- ingreso: verde
- egreso: rojo
- info: violeta
