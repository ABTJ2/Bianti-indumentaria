Cuando terminemos la reestructuración visual y funcional, recién ahí hacé una auditoría de limpieza.

Objetivo:
detectar archivos realmente inútiles o duplicados.

Importante:
- No borres archivos automáticamente.
- Primero entregame una lista dividida en:
  1. archivos activos
  2. archivos secundarios todavía útiles
  3. archivos candidatos a eliminar
  4. archivos que no deben borrarse por seguridad

Recién después de mi confirmación podés hacer limpieza.
