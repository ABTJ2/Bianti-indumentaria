# Referencias visuales BIANTI para OpenCode

Esta carpeta contiene las imágenes de referencia del diseño aprobado para BIANTI.

## Objetivo
OpenCode debe usar estas imágenes como referencia visual principal para:
- login admin
- catálogo público
- panel principal
- métricas
- categorías
- pedidos
- contabilidad
- estadísticas
- productos y nuevo producto como pantallas secundarias

## Regla principal
No incrustar las imágenes dentro del sistema.
No usar las imágenes como fondo directo.
No copiar las imágenes tal cual dentro del HTML.
Debe replicar el estilo con HTML, CSS y JavaScript.

## Cómo usar esta carpeta
1. Colocarla dentro del proyecto con el nombre `referencias_diseno`.
2. Pedirle a OpenCode que lea esta carpeta y `GUIA_VISUAL.md`.
3. Pedirle que combine estas referencias visuales con la carpeta `bianti_opencode_reestructura`, que contiene los requerimientos funcionales y técnicos.
