# Requerimientos no funcionales

## RNF-01 Diseño
El diseño debe ser moderno, limpio, elegante y alineado a BIANTI.

## RNF-02 Consistencia
Todas las pantallas admin deben usar el mismo layout.

## RNF-03 Usabilidad
La persona administradora debe poder entender rápidamente qué hacer.

## RNF-04 Seguridad básica
Las pantallas admin deben requerir login.

## RNF-05 Mantenibilidad
El código debe quedar ordenado y fácil de modificar.

## RNF-06 No romper datos
No se deben borrar datos ni modificar estructura de Supabase sin autorización.

## RNF-07 Rendimiento
Las consultas deben ser razonables y no cargar datos innecesarios.

## RNF-08 Responsive
Debe verse aceptable en escritorio y adaptable a pantallas menores.
