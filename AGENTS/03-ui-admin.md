# AGENTE 3 - UI/UX PANEL ADMIN

## Objetivo
Rediseñar el panel de administración para que se vea como un sistema real de gestión comercial.

## Archivos principales
- admin/_sidebar.html
- admin/login.html
- admin/dashboard.html
- admin/productos.html
- admin/nuevo-producto.html
- admin/categorias.html
- admin/pedidos.html
- admin/metricas.html
- admin/tracking.html
- admin/contabilidad.html
- admin/backups.html
- assets/css/admin.css

## Cambios obligatorios
- Eliminar emojis.
- Usar SVG icons profesionales.
- Mejorar sidebar.
- Mejorar separación visual de secciones.
- Mejorar botones.
- Mejorar tablas.
- Mejorar formularios.
- Mejorar cards.
- Mejorar títulos y subtítulos.
- Mejorar espaciados.
- Mejorar responsive.
- Mejorar estados visuales: activo, pendiente, vendido, error, vacío.

## Sidebar
Debe tener:
- Logo BIANTI claro.
- Secciones ordenadas.
- Íconos SVG.
- Estado activo visible.
- Botón cerrar sesión abajo.
- Diseño profesional.
- Nada de emojis.

## Páginas
Cada página debe tener:
- Título claro.
- Descripción breve.
- Acciones principales arriba.
- Contenido en cards o paneles.
- Tablas prolijas.
- Formularios bien agrupados.
- Mensajes vacíos bien diseñados.

## Reglas
- No tocar lógica de Supabase salvo que sea necesario.
- No eliminar funciones existentes.
- No romper login/logout.
- No romper navegación.
- No cambiar rutas sin actualizar enlaces.
- No eliminar IDs usados por JavaScript.

## Entrega esperada
Panel admin coherente, moderno y cómodo de usar.
