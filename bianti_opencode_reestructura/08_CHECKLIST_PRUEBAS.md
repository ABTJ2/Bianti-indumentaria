# Checklist de pruebas

## Login
- [ ] El login abre.
- [ ] Usuario correcto entra.
- [ ] Usuario incorrecto muestra error.
- [ ] El admin queda guardado en localStorage.
- [ ] Cerrar sesión funciona.

## Sidebar
- [ ] Aparece en todas las pantallas.
- [ ] Tiene solo 6 opciones.
- [ ] El orden es correcto.
- [ ] El activo se marca bien.
- [ ] No aparece Configuración.
- [ ] No aparece Backups.
- [ ] No aparece Tracking.
- [ ] No aparece Clientes.
- [ ] No aparece Consultas.

## Panel principal
- [ ] Carga sin errores.
- [ ] Muestra KPIs.
- [ ] Muestra últimas consultas.
- [ ] Muestra flujo de ventas.
- [ ] Accesos rápidos funcionan.

## Categorías
- [ ] Lista categorías.
- [ ] Abre modal.
- [ ] Crea categoría.
- [ ] Edita categoría.
- [ ] Visible funciona.
- [ ] Usa talles funciona.
- [ ] Orden funciona.
- [ ] Eliminar controla errores.

## Pedidos
- [ ] Lista pedidos.
- [ ] Filtra por estado.
- [ ] Busca por cliente/producto.
- [ ] Abre modal.
- [ ] Marca vendido.
- [ ] Marca no vendido.
- [ ] Vendido puede generar venta si está implementado.
- [ ] No vendido no impacta contabilidad.

## Contabilidad
- [ ] Carga ingresos.
- [ ] Solo cuenta ventas confirmadas.
- [ ] No cuenta pedidos en revisión.
- [ ] No cuenta no vendidos.
- [ ] Muestra gráficos.
- [ ] Muestra movimientos.

## Métricas
- [ ] Muestra consultas.
- [ ] Muestra ventas.
- [ ] Calcula conversión.
- [ ] Muestra productos consultados.
- [ ] Muestra productos vendidos.

## Estadísticas
- [ ] Muestra más vendido.
- [ ] Muestra más consultado.
- [ ] Muestra categoría más buscada.
- [ ] Muestra destacados automáticos.
- [ ] Muestra gráfico consultas/ventas/conversión.

## Catálogo público
- [ ] Sigue mostrando productos.
- [ ] Filtros funcionan.
- [ ] WhatsApp funciona.
- [ ] Consulta crea pedido o evento según lógica actual.
